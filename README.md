# Ticno VEX

مرحباً بك في مستودع مشروع Ticno VEX، موقع ويب تفاعلي مصمم خصيصاً لعرض المشاريع البرمجية.

## المميزات الرئيسية

- واجهة مستخدم عربية متجاوبة
- تصميم عصري مع تأثيرات حركية جذابة
- أقسام متعددة تشمل:
  - الصفحة الرئيسية
  - من نحن
  - المشاريع
  - اتصل بنا

## التقنيات المستخدمة

- React.js
- CSS3 مع متغيرات CSS
- HTML5
- JavaScript (ES6+)
